# 🌸 BloomSphere – AI-Powered Iris Flower Classification System

BloomSphere is a premium machine learning web application that demonstrates how Artificial Intelligence can classify flower species using botanical measurements. The project is built around the famous Iris Dataset and uses the K-Nearest Neighbors (KNN) algorithm to perform real-time species prediction directly inside the browser.

Designed with a modern glassmorphism-inspired interface, BloomSphere combines machine learning concepts, data visualization, and interactive user experience into a single educational platform.

---

## 🚀 Project Overview

BloomSphere allows users to enter flower measurements and instantly predict whether the flower belongs to:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

The application demonstrates the complete machine learning workflow including:

* Dataset Exploration
* Data Preprocessing
* Feature Scaling
* Train-Test Split
* K-Nearest Neighbors Classification
* Model Evaluation
* Real-Time Prediction

---

## ✨ Key Features

### 🌼 Intelligent Species Prediction

Predict Iris flower species using Sepal Length, Sepal Width, Petal Length, and Petal Width measurements.

### 🤖 K-Nearest Neighbors (KNN) Algorithm

Implements a client-side KNN classifier using Euclidean Distance calculations.

### 📊 Model Evaluation Dashboard

Displays important machine learning performance metrics:

* Accuracy
* Precision
* Recall
* F1 Score

### 🔍 Confusion Matrix Visualization

Provides a visual representation of model predictions versus actual classifications.

### 📚 Dataset Explorer

Explore Iris Dataset records with filtering and search capabilities.

### ⚡ Real-Time Inference

Predictions are generated instantly without requiring any backend server.

### 🎨 Premium User Interface

Modern dark-themed glassmorphism design with responsive layouts, animations, gradients, and interactive components.

## 📈 Evaluation Metrics

BloomSphere evaluates model performance using:

### Accuracy

Measures the percentage of correct predictions.

### Precision

Measures prediction quality for each class.

### Recall

Measures the model's ability to identify all relevant samples.

### F1 Score

Harmonic mean of Precision and Recall.

### Confusion Matrix

Provides class-wise performance analysis and prediction distribution.
