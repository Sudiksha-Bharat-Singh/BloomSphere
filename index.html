About BloomSphere

BloomSphere is an AI-powered flower classification system built using the Iris Dataset and the K-Nearest Neighbors (KNN) algorithm. It demonstrates supervised machine learning concepts including data preprocessing, training, prediction, and model evaluation.

How the KNN Classification Pipeline Works

1. Feature Preprocessing

User inputs are standardized in real-time using Z-score scaling calculated from dataset means and standard deviations, matching the model's training scale.

2. Proximity Search

The system calculates 4-dimensional Euclidean distances from the query vector to all stored training instances in client memory.

3. Neighborhood Voting

The engine selects the k = 5 closest instances. These neighbors cast category votes, resolving any splits via distance-based tie-breakers.
