Meet The Iris Species

The Iris Dataset contains three distinct flower species used for machine learning classification.

Class 1

Iris Setosa

Iris setosa

Known for its small petals and easy separability from other species.

Short Petals Wide Sepals Linearly Separable
Class 2

Iris Versicolor

Iris versicolor

A medium-sized species with balanced petal and sepal dimensions.

Medium Petals Moderate Sepals Feature Overlap
Class 3

Iris Virginica

Iris virginica

The largest Iris species with longer petals and broader measurements.

Long Petals Narrow Sepals Complex Boundary
Each species contains 50 samples in the Iris Dataset.
Pipeline Lifecycle

Machine Learning Workflow

01

Data Collection

Load the 150 Iris instances containing 4 numeric dimensions and target classes.

02

Preprocessing

Normalize inputs to ensure features with larger values do not bias distance calculations.

03

Train/Test Split

Partition data into Train/Test splits. Store training vectors as local references.

04

KNN Algorithm

Identify the $k$ nearest neighbors to the test vector using Euclidean distance.

05

Evaluation

Predict species based on majority voter category. Update metrics output.

06

Prediction

Deliver the final classifications dynamically to the prediction workspace inline.
